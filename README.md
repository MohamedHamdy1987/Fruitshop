# 🥦 VegERP – نظام إدارة تجارة الخضار والفواكه بالجملة

نظام متكامل لإدارة الموردين، المخزون، المبيعات (نقدي/آجل)، الفواتير التلقائية، الخزنة، الموظفين، الشركاء، والمحلات.

## التقنيات
- HTML5 / CSS3 / JavaScript (Vanilla)
- Supabase (PostgreSQL, Auth, RLS, Realtime)
- Chart.js

## التشغيل المحلي
1. انسخ المستودع
2. افتح `index.html` باستخدام Live Server
3. سجل حساب جديد (تجربة مجانية 14 يوم)
