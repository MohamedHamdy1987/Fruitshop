# 🍎 Market System Pro

نظام احترافي لإدارة تجارة الخضار والفواكه (سوق الجملة)  
مصمم خصيصًا للتجار لإدارة الفواتير، المبيعات، العملاء، الموردين، الخزنة، والترحيلات — في مكان واحد.

---

## 🚀 المميزات الرئيسية

### 📄 إدارة الفواتير
- إنشاء فواتير الموردين
- إضافة أصناف (طماطم – تفاح – ...)
- تتبع الكميات (مباع / متبقي / مرتجع)
- إغلاق تلقائي للفواتير

---

### 🛒 نظام مبيعات متكامل
يدعم 3 أنواع بيع:

- 💰 كاش
- 👤 آجل (عملاء)
- 🏬 محلات (تجار السوق)

---

### 👥 العملاء (كشف حساب ذكي)
- رصيد مبدئي
- يوميات تلقائية
- خصومات / قطعية
- حساب صافي لحظي

---

### 📋 الترحيلات (يومية العملاء)
- عرض يومي لكل عميل
- تفاصيل الأصناف
- تجميع تلقائي

---

### 🏬 محلات السوق
- نظام "لنا / لهم"
- تسجيل تعاملات بين التجار
- ترحيل تلقائي للعملاء عند البيع

---

### 💰 الخزنة
- تسجيل التحصيلات من العملاء
- تسجيل المصروفات
- صافي الخزنة

---

### 📊 Dashboard
- رسوم بيانية للمبيعات
- تحليل الأرباح

---

### 👷 الموظفين
- إدارة الموظفين
- صلاحيات (قابل للتطوير)

---

## 🔐 الأمان (RLS)

النظام يعتمد على:

> Row Level Security (RLS)

✔️ كل مستخدم يرى بياناته فقط  
✔️ نظام SaaS حقيقي  
✔️ بدون تسريب بيانات  

---

## 🧱 التقنيات المستخدمة

- HTML / CSS / JavaScript
- Supabase (Database + Auth + RLS)
- Chart.js (للرسوم البيانية)
- GitHub Pages (Hosting)
